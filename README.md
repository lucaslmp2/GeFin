## 📖 Sobre o Projeto

**GeFin** é uma aplicação web de front-end para gerenciamento de finanças pessoais. Permite ao usuário adicionar, editar, excluir e visualizar registros de crédito e débito de forma intuitiva. Todos os dados são salvos localmente no navegador, utilizando a API de `localStorage`, o que garante que as informações persistam entre as sessões.

Este projeto foi desenvolvido com foco em boas práticas de HTML semântico, CSS moderno e JavaScript modular, visando criar uma experiência de usuário fluida e uma base de código limpa e de fácil manutenção.

---

## ✨ Funcionalidades Principais

- **CRUD de Registros:** Crie, leia, atualize e exclua lançamentos financeiros.
- **Persistência de Dados:** As informações são salvas no `localStorage` do navegador.
- **Formulário em Modal:** Adicione e edite registros sem sair da tela principal, proporcionando uma experiência de usuário mais ágil.
- **Filtragem por Período:** Visualize os registros financeiros dentro de um intervalo de datas específico.
- **Resumo Financeiro:** Acompanhe o saldo anterior, o total de créditos, o total de débitos e o saldo final do período selecionado.
- **Confirmação de Exclusão:** Um alerta de confirmação impede a remoção acidental de dados.
- **Design Responsivo:** A interface se adapta a diferentes tamanhos de tela, de desktops a dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

- **HTML5:** Para a estrutura semântica do conteúdo.
- **CSS3:** Para a estilização, layout (Flexbox) e responsividade.
- **JavaScript (ES6+):** Para toda a lógica da aplicação, manipulação do DOM e interação com o `localStorage`.
- **Font Awesome:** Para os ícones utilizados na interface.

## 🚀 Como Executar o Projeto

Como este é um projeto puramente de front-end, não há necessidade de um servidor ou de processos de build complexos.

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/gerenciador-financeiro.git
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd gerenciador-financeiro
    ```
3.  **Abra o arquivo `table.html` no seu navegador de preferência.**

E pronto! A aplicação estará funcionando.
